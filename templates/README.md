# MikeOS Templates

Use these templates to keep cockpit output consistent.

- `daily-card.md` - one item for the daily brief.
- `worker-brief.md` - a bounded question sent to a specialist workspace.
- `worker-report.md` - the distilled result returned from a specialist workspace.

Templates are intentionally plain Markdown so they can be copied into `state/today.md`, `reports/`, or a worker prompt without tooling.
