# MikeOS Reports

Store generated cockpit reports here when a daily composition, machine-health summary, source-freshness review, or worker synthesis needs a durable artifact.

Reports should be distilled and link back to canonical systems. Do not store raw source evidence.

Suggested file names:

- `YYYY-MM-DD-daily-composition.md`
- `YYYY-MM-DD-worker-summary-<role>.md`
- `YYYY-MM-DD-machine-health.md`
- `YYYY-MM-DD-source-freshness.md`
