# Manual Worker Summaries

Put hand-authored or pasted `WorkerReport` summaries here when a specialist workspace has already returned a distilled result.

`scripts/build-today.ps1` looks for Markdown files in this folder and treats their presence as a cockpit signal. The script does not parse raw evidence and does not query live systems.

Do not store raw email, Teams, ADO, Home Assistant trace, attachment, or transcript bodies here.
